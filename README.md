# Task Manager CLI

A Python-based task manager application with:

- Task creation
- Task deletion
- Task completion tracking
- Priority support
- JSON storage
- Logging system
- CLI command support

---

# Features

- Interactive menu mode
- Command-line mode
- Persistent task storage
- Config-based settings
- Error handling and validation

---

# Run Interactive Mode

```bash
python main.py
```
---

# Run CLI Commands

```bash
python main.py add "task"
python main.py list
python main.py complete 1
python main.py delete 1
```
